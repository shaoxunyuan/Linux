Linux基础及应用教程实验：列出文件

一、实验目标

  1. 熟练掌握 ls 命令的多种使用方式，理解不同参数的功能与输出差异。
  2. 能够通过 ls 快速查看文件目录结构及文件属性。
  3. 掌握 ls 在系统文件管理与排错中的常见应用场景。

二、实验环境

  1. 硬件：机房 PC（已启用虚拟化功能）。
  2. 软件：VMware Workstation + CentOS 系统（预装，账号密码见桌面 “关于密码.txt”）。

三、实验内容与步骤

任务 1：查看当前目录文件

执行以下命令：

ls

观察 /home/njucm 目录下的文件与文件夹列表。
输出仅显示文件名，不包含详细信息。

任务 2：查看详细文件信息

执行：

ls -l

输出示例：

drwxr-xr-x 2 njucm njucm 4096 Oct 10 10:00 Documents
-rw-r--r-- 1 njucm njucm 512 Oct 23 09:30 notes.txt

说明：

  • 第一列：文件权限（如 drwxr-xr-x）
  • 第二列：硬链接数量
  • 第三、四列：所有者与所属组
  • 第五列：文件大小（单位为字节）
  • 第六列：最后修改时间
  • 第七列：文件或目录名称

任务 3：显示隐藏文件

执行：

ls -a

此命令会显示以 . 开头的隐藏文件，例如：.bashrc、.config 等。
这些文件通常用于保存系统或用户配置。

任务 4：查看根目录内容

执行：

ls /

查看系统根目录下的主要结构，包括 /home、/etc、/var、/usr 等目录。
这些目录分别存放用户文件、配置文件、日志文件与系统程序。

任务 5：按修改时间排序

执行：

ls -lt

该命令按最后修改时间降序排列，最新修改的文件会显示在最上方。
常用于分析最近修改的配置或日志文件。

任务 6：以人类可读格式显示文件大小

执行：

ls -lh

在 -l 的基础上，使用 -h 参数将文件大小以 KB/MB/GB 格式显示，便于阅读。

任务 7：组合参数查看详细且包含隐藏文件

执行：

ls -alh

此命令综合使用 -a、-l、-h 参数，常用于系统排错或详细文件审查。

任务 8：查看目录内容层级结构（扩展命令）

执行：

ls -R

显示当前目录及所有子目录的内容（递归显示）。
适合在复杂项目结构中快速查看目录组织情况。

四、实验报告要求

  1. 提供 ls、ls -l、ls -a、ls / 的执行结果截图（需包含命令输入与完整输出）。
  2. 对比 ls 与 ls -l 的输出差异，说明 -l 参数的作用及适用场景。
  3. 展示 ls -alh 的执行结果，并说明每列信息的含义。
  4. 举例解释隐藏文件（如 .bashrc、.vimrc）的作用。
  5. 执行 ls -lt 后，分析最近修改的文件用途。
  6. 比较 ls 与 ls -R 的输出差异，说明递归查看目录的意义。
  7. 使用 ls -lh /etc，观察分析系统配置文件的大小与更新时间。

✅ 提示：

  • ls 命令的参数可组合使用（如 ls -alh）。
  • 对于大目录，建议配合 grep、head、tail 等命令过滤查看。
  • 常用别名可添加到 ~/.bashrc 文件中，例如：
alias ll='ls -alh'
source ~/.bashrc
